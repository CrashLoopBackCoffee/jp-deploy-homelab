# AdGuard

## Purpose

DNS filtering within the homelab.

## Current role

AdGuard is under evaluation as part of the DNS path together with Unbound and dnsmasq.

## Key open question

Whether AdGuard should remain a distinct filtering layer, and how it should interact with local hostname resolution and recursive resolution.
